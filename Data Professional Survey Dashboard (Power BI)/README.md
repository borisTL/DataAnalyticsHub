# Data Professional Survey Dashboard (Power BI)

This project is an interactive Power BI dashboard based on survey data from data professionals.  
The report highlights salaries, job roles, programming languages, career entry difficulty, and overall job satisfaction.

## Dashboard Preview

![Dashboard Preview](screenshots/dashboard.png)

## Key Insights

- Distribution of respondents by country of residence  
- Average salary by job title  
- Favorite programming languages  
- Difficulty of breaking into the data field  
- Happiness with work–life balance and salary  
- Total number of respondents and other KPIs

## Tools and Skills

- Power BI Desktop  
- Power Query for data cleaning  
- DAX for calculated measures  
- Data modeling  
- Interactive report design
